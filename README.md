# BuildCored-Orcas-Day18
PWMSimulator — BUILDCORED ORCAS Day 18

What it does. It’s like an AI "brain" for your webcam that guesses how far away things are just by looking at them. It turns your video into a colorful map where bright colors mean "close" and dark colors mean "far."

Hardware concept. In the real world, robots and self-driving cars use a LiDAR sensor, which is basically a laser-powered ruler. Since we don't have a laser ruler, we are using this AI code to trick our normal camera into acting like a fancy 3D sensor.

Screen Recording. https://drive.google.com/file/d/1_q5drJcoMJ4O5JWbrsyfhetES2vx-CoX/view?usp=sharing

What I would do differently. I’d try to make it less "jittery" so the colors don't jump around so much when I move my hand. I’d also want to make it run faster, because right now it feels a little laggy, like a video game with a slow connection.
